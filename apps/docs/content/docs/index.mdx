# AI Chat Starter

A modern, production-ready template for building AI chat applications with Next.js, TypeScript, and a comprehensive monorepo structure.

## Features

- **Next.js 15 + React 19** - Latest React features with App Router
- **TypeScript** - Full type safety with strict configuration  
- **Monorepo** - Turborepo for optimal development experience
- **AI Ready** - Interfaces for OpenAI, Anthropic, and other providers
- **UI Components** - Radix UI + Tailwind CSS component library
- **Testing** - Vitest + Testing Library setup
- **Dev Tools** - ESLint, Prettier, Husky pre-commit hooks

## Architecture

```
ai-chat-starter/
├── apps/
│   ├── web/                 # Next.js chat application
│   └── docs/                # Documentation site
├── packages/
│   ├── ui/                  # Shared UI components
│   ├── types/               # TypeScript definitions
│   └── config/              # Shared configurations
```

## Getting Started

```bash
# Clone and install
git clone <your-repo>
cd ai-chat-starter
pnpm install

# Start development
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the chat interface.
