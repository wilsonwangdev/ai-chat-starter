# AI 聊天启动器

一个现代化、生产就绪的 AI 聊天应用模板，基于 Next.js、TypeScript 和完整的 monorepo 架构构建。

## 特性

- **Next.js 15 + React 19** - 最新的 React 特性和 App Router
- **TypeScript** - 严格的类型安全配置
- **Monorepo** - 使用 Turborepo 优化开发体验
- **AI 就绪** - 支持 OpenAI、Anthropic 等多种 AI 提供商
- **UI 组件** - 基于 Radix UI + Tailwind CSS 的组件库
- **测试** - Vitest + Testing Library 测试配置
- **开发工具** - ESLint、Prettier、Husky 预提交钩子

## 架构

```
ai-chat-starter/
├── apps/
│   ├── web/                 # Next.js 聊天应用
│   └── docs/                # 文档站点
├── packages/
│   ├── ui/                  # 共享 UI 组件
│   ├── types/               # TypeScript 类型定义
│   └── config/              # 共享配置文件
```

## 开始使用

```bash
# 克隆并安装
git clone <your-repo>
cd ai-chat-starter
pnpm install

# 启动开发
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看聊天界面。
